# ReFrog

Open source alternative to the WeCroak app: a quote journal app about death with a minimalist aesthetic.

## Features

- **Daily Quotes**: Quotes to remind you of your mortality to contemplate
- **Custom Quotes**: Add your own quotes to the rotation
- **Favorites**: Save favorite quotes to your journal
- **Notifications**: Schedule daily reminder quotes
- **Book Shelf**: Organize quotes by book
- **Dislike Filter**: Hide quotes you do not want to see again
